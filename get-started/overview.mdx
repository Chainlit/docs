## Key features

1. [Build fast:](/examples/openai-sql) Integrate seamlessly with an existing code base or start from scratch in minutes

2. [Copilot:](/deployment/copilot) Embed your chainlit app as a Software Copilot

3. [Data persistence:](/data-persistence/overview) Collect, monitor and analyze data from your users

4. [Visualize multi-steps reasoning:](/concepts/step) Understand the intermediary steps that produced an output at a glance

5. [Iterate on prompts:](/advanced-features/prompt-playground) Deep dive into prompts in the Prompt Playground to understand where things went wrong and iterate
