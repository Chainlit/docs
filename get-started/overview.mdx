Chainlit is an open-source Python package that makes it incredibly fast to build Chat GPT like applications with your own business logic and data.

## Key features

1. [Build fast:](/examples/openai-sql) Integrate seamlessly with an existing code base or start from scratch in minutes

2. [Data persistence:](/chat-data/overview) Leverage user generated data + feedback.

3. [Visualize multi-steps reasoning:](/observability-iteration/chain-of-thought) Understand the intermediary steps that produced an output at a glance

4. [Iterate on prompts:](/observability-iteration/prompt-playground) Deep dive into prompts in the Prompt Playground to understand where things went wrong and iterate
