---
title: "langchain_rename"
---
Useful to rename the LangChain tools/chains used in the agent and display more friendly author names in the UI.

### Parameters

<ParamField path="orig_author" type="str" required>
  The original LangChain chain/tool author name.
</ParamField>

### Returns

<ResponseField name="author" type="str" required>
  The renamed author
</ResponseField>

<RequestExample>
```python Code Example
from langchain import OpenAI, LLMMathChain
import chainlit as cl

@cl.langchain_rename
def rename(orig_author: str):
    rename_dict = {
        "LLMMathChain": "Albert Einstein"
    }

    return rename_dict.get(orig_author, orig_author)

@cl.langchain_factory
def load():
    llm = OpenAI(temperature=0)
    llm_math = LLMMathChain.from_llm(llm=llm)

    return llm_math
```
</RequestExample>
