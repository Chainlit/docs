---
title: "GenerationMessage"
---

`GenerationMessage` is used in the [ChatGeneration](/api-reference/generation/chat-generation) class to represent a message.

## Attributes

<ParamField path="content" type="str" optional>
  The content of the message.
</ParamField>

<ParamField path="role" type="str" optional>
  The role of the message, such as "system", "assistant" or "user".
</ParamField>

<ParamField path="name" type="str" optional>
  Only used for OpenAI's function message.
</ParamField>
