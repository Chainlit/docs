---
title: get_element
---

Get information for a specific element.

### Parameters

<ParamField path="conversation_id" type="str">
  The ID of the conversation to which the element belongs.
</ParamField>
<ParamField path="element_id" type="str">
  The ID of the element to retrieve information for.
</ParamField>

### Returns

<ResponseField name="element" type="Optional[ElementDict]" required>
  The element information if found, None otherwise.
</ResponseField>

### Example

```python Code Example
from chainlit.client.cloud import chainlit_client

conversation_id = "12345"
element_id = "54321"

element = await chainlit_client.get_element(conversation_id, element_id)

if element:
    print(f"Element ID: {element['id']}")
    print(f"Type: {element['type']}")
    print(f"Name: {element['name']}")
else:
    print("No element found.")
```
