---
title: delete_conversation
---

Delete a conversation.

### Parameters

<ParamField path="conversation_id" type="str">
  The ID of the conversation to delete.
</ParamField>

### Returns

<ResponseField name="success" type="bool" required>
  True if the conversation was successfully deleted, False otherwise.
</ResponseField>

### Example

```python Code Example
from chainlit.client.cloud import chainlit_client

conversation_id = "12345"

success = await chainlit_client.delete_conversation(conversation_id)

if success:
    print("Conversation deleted.")
else:
    print("Failed to delete conversation.")
```
