---
title: delete_message
---

Delete a message.

### Parameters

<ParamField path="message_id" type="str">
  The ID of the message to delete.
</ParamField>

### Returns

<ResponseField name="success" type="bool" required>
  True if the message was successfully deleted, False otherwise.
</ResponseField>

### Example

```python Code Example
from chainlit.client.cloud import chainlit_client

message_id = "12345"

success = await chainlit_client.delete_message(message_id)

if success:
    print("Message deleted.")
else:
    print("Failed to delete message.")
```
