---
title: get_conversation
---

Get information for a specific conversation.

### Parameters

<ParamField path="conversation_id" type="str">
  The ID of the conversation to retrieve information for.
</ParamField>

### Returns

<ResponseField name="conversation" type="ConversationDict" required>
  The conversation information if found.
</ResponseField>

### Example

```python Code Example
from chainlit.client.cloud import chainlit_client

conversation_id = "12345"

conversation = await chainlit_client.get_conversation(conversation_id)

if conversation:
    print(f"Conversation ID: {conversation['id']}")
    print(f"Created At: {conversation['createdAt']}")
else:
    print("No conversation found.")
```
