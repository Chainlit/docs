---
title: Formatted Prompts
---

The formatted prompts view allows you to preview the full text that is sent to the LLM for completion.

This screen enables you to edit the current LLM inputs and get a completion, enabling you to debug your app and to test and iterate on your prompts.

![Formatted Prompt](/images/pp-formatted-prompt.png)

## Setup

If you did not already setup the generation object, go [here](/advanced-features/prompt-playground/setup).
Once this is done, open the Prompt Playground.
