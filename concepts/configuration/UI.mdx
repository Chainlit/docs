---
title: UI
---

Chainlit UI can be customized by updating your `config.toml`.

### Options

<ParamField path="name" type="str" default="My Chatbot">
  The name of both the application and the chatbot.
</ParamField>

<ParamField path="description" type="str" optional>
  The content of the `<meta name="description">` of the application.
</ParamField>

<ParamField path="hide_cot" type="bool" default={false}>
  You can hide the [Chain of Thought](/concepts/chain-of-thought) UI.
</ParamField>

<ParamField path="default_collapse_content" type="bool" default={true}>
  When handling large text content we collapse it for keeping the threads concise.
  You can disable manually disable this behavior.
</ParamField>

<ParamField path="default_expand_message" type="bool" default={false}>
  Sub-messages are hiden by default, you can "expand" the parent message to show those messages.
  Toggling this setting will display the sub-messages by default.
</ParamField>

<ParamField path="github" type="str" optional>
  Passing this option will display a Github-shaped link.
  If not passed we will display the link to Chainlit repo.
</ParamField>

### Default configuration

```toml
[UI]
name = "My Chatbot"
description = ""
hide_cot = false
default_collapse_content = true
default_expand_message = false
github = ""
```
