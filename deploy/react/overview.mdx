The `@chainlit/react-client` package allows you to seamlessly integrate Chainlit's backend capabilities with your React applications. This integration enables real-time communication, state management, and various chat functionalities using React hooks and the Chainlit API client.

It is composed of several core features and hooks that make managing chat sessions, messages, authentication, and custom UI elements easier.
