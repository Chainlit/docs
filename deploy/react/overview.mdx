title: "Overview"
---

The `@chainlit/react-client` package allows you to seamlessly integrate Chainlit's backend capabilities with your React applications, enabling real-time communication, state management, and various chat functionalities using React hooks and the Chainlit API client.

It consists of several core features and hooks that simplify managing chat sessions, messages, authentication, and custom UI elements.
